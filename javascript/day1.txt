자바스크립트의 특징
1.인터프리터 언어 - 한라인씩 해석
2.동적 프로토타입 기반 객체 지향 언어
3.동적 타입 언어
4.함수가 일급 객체다.- 함수에 함수를 인수로 넘길 수 있다.
5.함수가 클로저를 정의한다. - 변수 은닉,영속성 보장

기술적 요소
1.ecmascript core
2.클라이언트의 고유한 기술 요소 - api(drag drop blob canvas ...)
3.서버 측의 기술 요소로도 활용 - node.js/rhino ...

자바스크립트 싱행법
1. 브라우저 console (f12)
2. html 문서에 삽입하여 브라우저로 실행
 > head 태그 내에 전역변수, 함수 선언 
   body의 요소를 참조하거나, 사용하는 자바스크립트 실행 문장은 오류 발생

 > 즉시 실행 문장은 script 태그 내에 사용
 > 보통은 html문서와 자바스크립트를 분리하여 사용
   <script src="./~~~.html"></script>
